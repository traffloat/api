Macro fluent::fluent_args[−][src]

macro_rules! fluent_args {
    ($($key : expr => $value : expr), *) => { ... };
}
Expand description

A helper macro to simplify creation of FluentArgs.

Example

use fluent::fluent_args;

let mut args = fluent_args![
    "name" => "John",
    "emailCount" => 5
];
